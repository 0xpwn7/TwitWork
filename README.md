# TwitWork
> Monitor twitter stream.

![TwitWork](https://i.imgur.com/uyNHILq.jpg)

## Installation

```sh
git clone https://github.com/atmoner/TwitWork.git
cd TwitWork
npm install
```
Open `renderer.js` and edit your variable Twitter api.
```
npm start
```
